第 3 课：偏导数与高阶导数

Partial Derivatives & Higher-Order Derivatives

核心洞察

偏导数就是把多元函数当作"只有一个变量在动，其他都冻结"来处理。你不需要学任何新的求导规则——所有一元微分的技术都能用，每次只对一个变量求导就行了。

0. 热身：为什么要分开求？

设想你站在一座山上。山的高度是 $z = f(x,y)$。你想知道：

  • 朝正东方向（只改变 $x$），山坡有多陡？→ 这就是 $\partialf/\partialx$
  • 朝正北方向（只改变 $y$），山坡有多陡？→ 这就是 $\partialf/\partialy$

偏导数独立地度量每个坐标方向的变化率。

1. 偏导数的定义

定义 — Partial Derivative

Let $n \ge 2$ denote a natural number, let $D \subseteq \mathbb{R}^n$ have non-empty interior, and let $w = (w_1, w_2, ..., w_n)$ be an interior point of $D$. If $f : D \to \mathbb{R}$ is a function of the independent variables $x_1, x_2, ..., x_n$, then the partial derivative of $f$ with respect to $x_j$ at $w$ is defined to be

$\partialf/\partialx_j (w) := lim_{h\to0} (f(w_1, ..., w_j₋_1, w_j + h, w_j₊_1, ..., w_n) − f(w_1, ..., w_j₋_1, w_j, w_j₊_1, ..., w_n)) / h$,

provided this limit exists.

例 1：$f(x,y) = x^2 + y^2$
对 $x$ 求偏导（把 $y$ 当常数）：$\partialf/\partialx = 2x + 0 = 2x$
对 $y$ 求偏导（把 $x$ 当常数）：$\partialf/\partialy = 0 + 2y = 2y$
例 2：$f(x,y) = x^2y^3 - 2xy^2 + 3y$
对 $x$ 求偏导：$\partialf/\partialx = 2xy^3 - 2y^2 + 0$（把 $y$ 当常数，$3y$ 的导数是 0）
对 $y$ 求偏导：$\partialf/\partialy = 3x^2y^2 - 4xy + 3$（把 $x$ 当常数，$-2xy^2$ 对 y 求导得 $-4xy$）

2. 偏导数的运算性质

定理 — 偏导数的代数性质

以下性质可以直接从一元导数继承：

  • 线性性（linearity）：$\partial(f+g)/\partialx⯋ = \partialf/\partialx⯋ + \partialg/\partialx⯋$
  • 数乘：$\partial(cf)/\partialx⯋ = c \cdot \partialf/\partialx⯋$（$c$ 为常数）
  • 乘积法则（product rule）：$\partial(fg)/\partialx⯋ = (\partialf/\partialx⯋)g + f(\partialg/\partialx⯋)$
  • 商法则（quotient rule）：$\partial(f/g)/\partialx⯋ = [(\partialf/\partialx⯋)g - f(\partialg/\partialx⯋)] / g^2$（当 $g \neq 0$）

也就是说，你不需要任何新技巧。每次只对一个变量求导，另一个看都不看。

3. 记号大全

偏导数有多种常用记号——碰到哪种都别慌：

一阶偏导数：
对 $x⯋$：$\partialf/\partialx⯋$、$fx⯋$、$D⯋f$
二阶纯偏导数：
对 $x⯋$ 求两次：$\partial^2f/\partialx⯋^2$、$fx⯋x⯋$、$D⯋⯋f$
二阶混合偏导数：
先对 $x⯊$ 再对 $x⯋$：$\partial^2f/\partialx⯋\partialx⯊$、$fx⯋x⯊$、$D⯋⯊f$
注意顺序：$\partial^2f/\partialx\partialy$ 表示先对 $y$ 再对 $x$（从右往左读，因为最右边的 $\partialy$ 在内层，先操作）。
混合偏导数的顺序：$fxy$ 和 $\partial^2f/\partialy\partialx$ 的书写顺序是颠倒的！前者 $fxy$ 表示先 $x$ 后 $y$。看仔细了——考试中写错了顺序会很惨。

4. 高阶偏导数

例：$f(x,y) = x^3y + e^{xy}$

一阶：
$fx = 3x^2y + ye^{xy}$
$fy = x^3 + xe^{xy}$

二阶纯：
$fxx = \partial/\partialx(3x^2y + ye^{xy}) = 6xy + y^2e^{xy}$
$fyy = \partial/\partialy(x^3 + xe^{xy}) = x^2e^{xy}$

二阶混合：
$fxy = \partial/\partialy(fx) = 3x^2 + e^{xy} + xye^{xy}$
$fyx = \partial/\partialx(fy) = 3x^2 + e^{xy} + xye^{xy}$
注意到 $fxy = fyx$——这不是巧合。

5. Clairaut 定理

定理 — Clairaut's Theorem

Suppose that two mixed $n$-th ordered partial derivatives of a function $f$ involve the same differentiations, but in different orders. If those partial derivatives are continuous at a point $p$, and if $f$ and all partial derivatives of $f$ of order less than $n$ are continuous in a neighbourhood of $p$, then the two mixed $n$-th ordered partial derivatives of $f$ at $p$ are equal.

25-Q1a-i 考试直接问了 Clairaut 定理的条件和等式成立的前提。
关键理解

Clairaut 定理说"混合偏导数通常相等"，前提条件是它们在该点连续。如果这个条件不满足，混合偏导数可能存在但不相等。

6. 反例：当 Clairaut 失效时

下面的经典反例说明连续性条件是必需的：

定义：
$f(x,y) = xy(x^2-y^2)/(x^2+y^2)$ 当 $(x,y) \neq (0,0)$，且 $f(0,0) = 0$。

计算结果：
$fxy(0,0) = 1$，但 $fyx(0,0) = -1$
两者不相等！这是因为混合偏导数在原点不连续。

7. 偏导数与连续性之间的关系

定理 — 偏导数的连续性与函数的连续性

如果 $f$ 的所有一阶偏导数在点 $w$ 的某个邻域内存在且有界，那么 $f$ 在 $w$ 处连续。

但反过来不成立：偏导数存在不能保证函数连续。

常见误区：在一元微积分中，"可导必连续"。在多元中，偏导数都存在不保证连续！原因是偏导数只检验了沿着坐标轴方向的行为，无法捕捉沿其他路径可能出现的问题。
反例 — 偏导数存在但不连续：
定义 $f(x,y) = 0$ 当 $xy = 0$，否则 $f(x,y) = 1$。
在原点：(a) 沿 x-轴，$f(x,0) = 0$ → $fx(0,0) = 0$；同理 $fy(0,0) = 0$。两个偏导数都存在！
但是 (b) 沿直线 $y = x$（$x \neq 0$），$f(x,x) = 1$，而 $f(0,0) = 0$。极限不等于函数值，函数不连续。

课堂测验

Q1. 设 $f(x,y) = ln(x^2 + y)$，则 $\partialf/\partialx$ 是？
A. $1/(x^2 + y)$
B. $2x/(x^2 + y)$
C. $2x/(x^2 + y^2)$
D. $(2x + 1)/(x^2 + y)$
正确答案：B
把 $y$ 当常数，对 $x$ 求导：$\partial/\partialx[ln(x^2 + y)] = (2x)/(x^2 + y)$。别忘了链式法则——对 $ln$ 求导得 $1/(x^2+y)$，再乘以括号内部对 $x$ 的导数 $2x$。
Q2.（考试题型）对于 $f(x,y) = x^3e^{2y} + y^2 sin x$，$fxy$ 在 $(0,0)$ 处等于多少？
A. 0
B. 1
C. 6
D. 不存在
正确答案：A — 0
先求 $fx = 3x^2e^{2y} + y^2 cos x$（把 $y$ 当常数）。
再对 $y$ 求偏导得 $fxy = 6x^2e^{2y} + 2y cos x$（把 $x$ 当常数，注意 $e^{2y}$ 对 y 求导要乘以 2）。
代入 $(0,0)$：$6\cdot0\cdote^0 + 0 = 0$。
你也可以先求 $fy$ 再求 $fyx$，根据 Clairaut 定理（偏导数在原点连续，确实满足条件）应该得到同样的答案 0。
Q3. Clairaut 定理断言 $fxy = fyx$ 在什么条件成立？
A. 只要 $f$ 可微
B. 只要 $f$ 连续
C. 当 $fxy$ 和 $fyx$ 在该点连续
D. 无条件成立，混合偏导数总是相等
正确答案：C
这正是 Clairaut 定理的核心——混合偏导数相等的前提是它们在考虑的点处连续。教科书上的经典反例（$f(x,y) = xy(x^2-y^2)/(x^2+y^2)$）说明当连续性不满足时，$fxy$ 和 $fyx$ 可以不同。
25-Q1a-i
Q4. 以下哪个说法是正确的？
A. 多元函数偏导数都存在，则函数一定连续
B. 多元函数偏导数都存在且偏导数有界，则函数连续
C. 多元函数偏导数都连续，则函数连续
D. B 和 C 都正确
正确答案：D
偏导数存在且有界（有界性在紧集上由连续导出）蕴含函数连续。偏导数都连续当然更强——自然蕴含函数连续。但仅仅偏导数存在（无界或无连续性保证）不足以保证连续性，上面那节课的反例已经说明了这一点。

本课要点总结

  • Partial derivative = 固定其他变量，对单一变量使用一元导数法则。
  • 偏导数记号多样——$\partialf/\partialx⯋$、$fx⯋$、$D⯋f$——看到哪种都别慌。注意混合偏导数的顺序约定。
  • Clairaut's Theorem：混合偏导数在其连续处相等。连续性是关键条件。
  • 偏导数存在 ≠ 函数连续（这和一元微积分有本质区别！）。偏导数只管坐标轴方向，管不了其他方向。
  • 实际操作就是一次又一次地使用一元求导法则——每次只动一个变量。
课外探索建议

用 Clairaut 定理验证 $f(x,y) = arctan(y/x)$ 的混合偏导数相等（在其定义域的任意点上）。然后思考：为什么 $f(x,y) = arctan(y/x)$ 在 $x = 0$ 上没定义？这和偏导数有什么关系？如果你需要训练更多高阶混合偏导数的计算题（考试每年必有一道），随时找我。
