第 4 课：链式法则与隐函数求导

Chain Rule & Implicit Differentiation

核心洞察

链式法则是整个 MVA 中使用频率最高的技巧。它告诉你：当函数的自变量本身也是函数时，导数怎么顺着"依赖链"传递下去。一旦掌握，隐函数求导、变量替换、偏微分方程求解都是它的自然延伸。

0. 热身：一元链式法则

先回忆老朋友：

一元链式法则：
若 $h(x) = u(v(x))$，则
$h'(x) = u'(v(x)) \cdot v'(x)$
或者写成 $dh/dx = (du/dv)(dv/dx)$。

多元的推广遵循同样的直觉："总变化率 = 沿着每条路径的变化率之和"。

1. 简单情形：自变量只有一个

定理 — 链式法则（最简单情形）

设 $f(u,v)$ 是二元函数，而 $u = u(t)$ 和 $v = v(t)$ 都是一元函数。那么复合函数 $F(t) = f(u(t), v(t))$ 的导数为：

$dF/dt = (\partialf/\partialu)(du/dt) + (\partialf/\partialv)(dv/dt)$

用中文说：把 $f$ 对每个自变量求偏导，乘以那个自变量对 $t$ 的导数，然后全部加起来。

树图法（Tree Diagram）

画一棵"依赖树"可以帮你理清思路：

    f
   / \
  u   v
  |   |
  t   t

从 $f$ 到 $t$ 有两条路径：f → u → t 和 f → v → t。每条路径上相乘，然后路径之间相加。这就是链式法则的直观：所有路径贡献之和。

2. 一般链式法则

定理 — Chain Rule

Let $m$ and $n$ denote two natural numbers. Suppose that $f$ is a function of $n$ real variables, $x_1, x_2, ..., x_n$, with continuous partial derivative with respect to $x_i$ throughout a non-empty open region $R \subseteq \mathbb{R}^n$, for all $i \in {1, 2, ..., n}$, each $x_j$ is a function of $m$ real variables, $t_1, t_2, ..., t_m$, with continuous partial derivative with respect to $t_i$ throughout a non-empty open set $D \subseteq \mathbb{R}ᵐ$, for all $i \in {1, 2, ..., m}$, and ${(x_1(t_1,...,t_m), ..., x_n(t_1,...,t_m)) \in \mathbb{R}^n : (t_1,...,t_m) \in D}$ is a subset of $R$. For all $i \in {1, 2, ..., m}$, the partial derivative of $f$ with respect to $t_i$ is given by

$\partialf/\partialt_i = (\partialf/\partialx_1)(\partialx_1/\partialt_i) + (\partialf/\partialx_2)(\partialx_2/\partialt_i) + ... + (\partialf/\partialx_n)(\partialx_n/\partialt_i)$

口头表述：把 $f$ 对每个"中间变量" $x⯋$ 的偏导数，乘以那个中间变量对目标变量 $t⯊$ 的偏导数，全部求和。

3. 完整计算示例

题目：设 $f(u,v,r) = ue^{vr}$，其中 $u = s^2t$，$v = st^2$，$r = s^2t^2$。求 $\partialf/\partials$ 和 $\partialf/\partialt$。
步骤 1：求偏导数。
$fu = e^{vr}$
$fv = ure^{vr}$
$fr = uve^{vr}$
步骤 2：求中间变量对 s 的偏导数。
$\partialu/\partials = 2st$
$\partialv/\partials = t^2$
$\partialr/\partials = 2st^2$
步骤 3：应用链式法则。
$\partialf/\partials = fu\cdotus + fv\cdotvs + fr\cdotrs$
$= e^{vr}\cdot2st + ure^{vr}\cdott^2 + uve^{vr}\cdot2st^2$
$= e^{vr}t(2s + urt + 2uvst)$
步骤 4：同理求 ∂f/∂t（把 s 当常数）。
$\partialu/\partialt = s^2$，$\partialv/\partialt = 2st$，$\partialr/\partialt = 2s^2t$
$\partialf/\partialt = e^{vr}\cdots^2 + ure^{vr}\cdot2st + uve^{vr}\cdot2s^2t$
$= e^{vr}s(s + 2urt + 2uvst)$
计算技巧：像 $ue^{vr}$ 这种函数，在求 $fu$ 时把 $e^{vr}$ 当常数（因为 $v$ 和 $r$ 不依赖于 $u$），在求 $fv$ 时把 $u$ 和 $r$ 当常数。保持清晰的"中间变量思维"可以避免混乱。

4. 隐函数求导 — 第一部分（二元情形）

22-Q1a25-Q1b

回想一下一元微积分：$x^2 + y^2 = 1$ 定义了 $y$ 作为 $x$ 的隐函数。在多元中我们有系统性的处理方法。

定理 — Implicit Differentiation (Part I)

Let $f$ be a function of two variables $x$ and $y$, and suppose that the partial derivatives of $f$ are continuous throughout a non-empty open region $R \subseteq \mathbb{R}^2$. If $f(x, y) = 0$ determines, implicitly, $y$ as a differentiable function of $x$, then

$\partialf/\partialy \cdot dy/dx = −\partialf/\partialx$

这个公式怎么来的？对 $f(x, y(x)) = 0$ 两边关于 $x$ 求全导数，用链式法则：

$d/dx [f(x, y(x))] = 0$
$\partialf/\partialx \cdot dx/dx + \partialf/\partialy \cdot dy/dx = 0$
$fx \cdot 1 + fy \cdot (dy/dx) = 0$
$⇒ dy/dx = -fx / fy$
例子：方程 $y^4 + 3y - 4x^3 - 5x = 1$ 隐式定义了 $y = y(x)$。求 $dy/dx$。

解法：令 $f(x,y) = y^4 + 3y - 4x^3 - 5x - 1 = 0$
$fx = -12x^2 - 5$
$fy = 4y^3 + 3$
$dy/dx = -fx/fy = -(-12x^2 - 5)/(4y^3 + 3) = (12x^2 + 5)/(4y^3 + 3)$

5. 隐函数求导 — 第二部分（三元情形）

定理 — Implicit Differentiation (Part II)

Let $f$ be a function of three variables $x$, $y$ and $z$, and suppose that the partial derivatives of $f$ are continuous throughout a non-empty open region $R \subseteq \mathbb{R}^3$. If $f(x, y, z) = 0$ determines, implicitly, $z$ as a function of $x$ and $y$ whose partial derivatives are continuous, then

$\partialf/\partialz \cdot \partialz/\partialx = −\partialf/\partialx$ and $\partialf/\partialz \cdot \partialz/\partialy = −\partialf/\partialy$

推导逻辑完全相同——对 $f(x, y, z(x,y)) = 0$ 关于 $x$（或 $y$）求偏导，应用链式法则。

例子：方程 $x^2 + y^2 + z^2 - 1 = 0$ 定义单位球面。求 $\partialz/\partialx$ 和 $\partialz/\partialy$。

$f(x,y,z) = x^2 + y^2 + z^2 - 1$
$fx = 2x$，$fy = 2y$，$fz = 2z$
$\partialz/\partialx = -2x/(2z) = -x/z$
$\partialz/\partialy = -2y/(2z) = -y/z$
注意：当 $z = 0$（赤道上）时，这些偏导数不存在——因为球面在赤道处有垂直的切平面。

6. 偏微分方程与变量替换

为什么 L2 就要知道 PDE？

Partial Differential Equations（PDEs，偏微分方程）是整个物理学的基础——热传导方程、波动方程、拉普拉斯方程、薛定谔方程——全是 PDE。考试中常见的题型是：通过 change of variables（变量替换）将某个 PDE 化简为标准形式。

22-Q1a25-Q1b 关于用新变量表达偏导数的结构化题目。
典型考试题型：设 $u = x + y$，$v = x - y$。将 $\partialf/\partialx$ 和 $\partialf/\partialy$ 用 $\partialf/\partialu$ 和 $\partialf/\partialv$ 表示。

步骤：由 $u = x + y, v = x - y$ 得 $x = (u+v)/2, y = (u-v)/2$。
应用链式法则：
$\partialf/\partialx = (\partialf/\partialu)(\partialu/\partialx) + (\partialf/\partialv)(\partialv/\partialx) = (\partialf/\partialu)\cdot1 + (\partialf/\partialv)\cdot1 = fu + fv$
$\partialf/\partialy = (\partialf/\partialu)(\partialu/\partialy) + (\partialf/\partialv)(\partialv/\partialy) = (\partialf/\partialu)\cdot1 + (\partialf/\partialv)\cdot(-1) = fu - fv$

这个技巧在后续课程中会反复用到——解波动方程、拉普拉斯方程等 PDE 时，选择合适的变量替换是整个求解过程的关键第一步。

课堂测验

Q1. 设 $f(u,v) = u^2 + v^2$，其中 $u = cos t$，$v = sin t$。用链式法则求 $df/dt$。
A. $0$
B. $2 cos t + 2 sin t$
C. $-2 cos t sin t$
D. $2(cos^2 t - sin^2 t)$
正确答案：A — 0
$fu = 2u$，$fv = 2v$。
$du/dt = -sin t$，$dv/dt = cos t$。
链式法则：$df/dt = fu\cdotdu/dt + fv\cdotdv/dt = 2u\cdot(-sin t) + 2v\cdotcos t = 2 cos t\cdot(-sin t) + 2 sin t\cdotcos t = -2 sin t cos t + 2 sin t cos t = 0$。
从几何角度看：$f(u,v) = u^2+v^2$ 在单位圆上恒为 1（因为 $cos^2t + sin^2t = 1$），常数函数的导数为 0。两种方法完美对应！
Q2. 方程 $x^2y + e^{y} - x = 0$ 隐式定义了 $y = y(x)$。在点 $(1,0)$ 处，$dy/dx$ 的值为？
A. $1$
B. $-1$
C. $1/2$
D. $0$
正确答案：A
设 $f(x,y) = x^2y + e^{y} - x$。
$fx = 2xy - 1$，$fy = x^2 + e^{y}$
在 $(1,0)$：$fx = 2\cdot1\cdot0 - 1 = -1$，$fy = 1^2 + e^0 = 2$
$dy/dx = -fx/fy = -(-1)/2 = 1$
先验证点(1,0)确实在曲线上：$1^2\cdot0 + e^0 - 1 = 0 + 1 - 1 = 0$。是的！
Q3.（考试题型）设 $w = f(x,y)$，其中 $x = r cos θ$，$y = r sin θ$。则 $\partialw/\partialθ$ 等于？
A. $fx cos θ + fy sin θ$
B. $-fx cos θ - fy sin θ$
C. $-fx r sin θ + fy r cos θ$
D. $fx r sin θ - fy r cos θ$
正确答案：C
应用链式法则：$\partialw/\partialθ = fx\cdot(\partialx/\partialθ) + fy\cdot(\partialy/\partialθ)$
$\partialx/\partialθ = -r sin θ$，$\partialy/\partialθ = r cos θ$
所以 $\partialw/\partialθ = fx\cdot(-r sin θ) + fy\cdot(r cos θ) = -r sin θ fx + r cos θ fy$
这是极坐标变换链式法则的经典考题，考试中反复出现。
Q4.（考试题型）设 $u = x - ct$，$v = x + ct$。若定义 $F(u,v) = f(x,t)$，则 $ftt - c^2fxx$ 用 $F$ 的偏导数怎么表示？
A. $Fuu + Fvv$
B. $4c^2Fuv$
C. $-4c^2Fuv$
D. $4Fuv$
正确答案：C — -4c²Fuv
这是波动方程（wave equation）的经典变量替换。链式法则：
$fx = Fu + Fv$ → $fxx = Fuu + 2Fuv + Fvv$
$ft = c(Fv - Fu)$ → $ftt = c^2(Fuu - 2Fuv + Fvv)$
代入：$ftt - c^2fxx = c^2(Fuu - 2Fuv + Fvv) - c^2(Fuu + 2Fuv + Fvv) = -4c^2Fuv$
波动方程 $ftt = c^2fxx$ 等价于 $Fuv = 0$，解得 $F(u,v) = φ(u) + ψ(v)$——d'Alembert 解。

本课要点总结

  • Chain Rule 的核心：每条依赖路径上的乘积之和。用 tree diagram 辅助思考。
  • 一般形式：$\partialf/\partialt⯊ = Σ⯋ (\partialf/\partialx⯋)(\partialx⯋/\partialt⯊)$——对每次出现在复合路径上的中间变量做贡献求和。
  • Implicit differentiation：$dy/dx = -fx/fy$（二元）和 $\partialz/\partialx = -fx/fz$（三元）——分子负号，分母是"目标变量"对应的偏导数。
  • Change of variables 是解 PDE 的第一步，也是考试的固定项目——用新偏导数表达旧偏导数。
  • 链式法则在后续的每一个话题中都会用到：方向导数、梯度、泰勒展开、极值、约束优化——无处不用。
课外探索建议

尝试用链式法则推导笛卡尔和极坐标下的拉普拉斯方程 $\partial^2f/\partialx^2 + \partial^2f/\partialy^2$（需要两次应用链式法则）。另外，问自己一个"反向"问题：如果已知 $fu$ 和 $fv$，能不能反解出 $fx$ 和 $fy$？这和变量替换的 Jacobian 矩阵有关，是下一课的内容。需要更多练习的话直接问我。
