第 11 课：向量微积分 — Vector Calculus

三大定理：Green's, Stokes', Divergence Theorems

DEFINITION

向量微积分（Vector Calculus）是多元微积分中最高潮的部分——它建立了曲线积分、曲面积分和区域积分之间的深层联系。三个核心定理——Green 定理、Stokes 定理、散度定理——本质上是同一个思想的三个版本：边界上的积分 = 内部某种「导数」的积分。

这部分内容在考试中约占 30-40% 的分数，是区分高分和及格的关键。We seriously need to get this right.

第一部分：保守场与路径无关性

核心问题

给定一个向量场 $F$，什么时候 $\int_C F\cdotdr$ 只依赖于起点和终点，而与具体走哪条路无关？

答案：当 $F$ 是保守场（conservative vector field）时。

DEFINITION — Conservative Vector Field / 保守向量场

Let $W$ denote a subset of $\mathbb{R}^3$. A $C^1$ vector field $F : W \to \mathbb{R}^3$ is said to be conservative if there exists a scalar function $φ : W \to \mathbb{R}$ such that $F = \nablaφ$.

此时对任意从 A 到 B 的曲线 C：

$\int_C F\cdotdr = φ(B) - φ(A)$

这就是微积分基本定理的向量版本——线积分不依赖于路径！

THEOREM — 保守场的判定条件

在二维情况下，如果 $F = \langle P(x,y), Q(x,y)\rangle$，那么 $F$ 是保守场的一个必要条件是：

$\partial P/\partial y = \partial Q/\partial x$

在三维情况下，条件是 $curl F = 0$。

如果区域是单连通的（simply connected，没有「洞」），那么这同时也是充分条件。

常见陷阱： $curl F = 0$ 不一定意味着 $F$ 就是保守场！经典反例：$F = \langle -y/(x^2+y^2), x/(x^2+y^2)\rangle$，它的 $curl = 0$（在定义域内），但绕原点一圈的线积分 = $2\pi$ ≠ 0。问题出在原点——那里有一个洞，区域不是单连通的。考试中如果你声称 curl F = 0 → F 保守，一定要提一下「且区域单连通」这个前提。

第二部分：散度与旋度

三个基本微分算子

梯度（Gradient）： 标量 → 向量

$\nabla f = \langle\partial f/\partial x, \partial f/\partial y, \partial f/\partial z\rangle$

散度（Divergence）： 向量 → 标量（某点「发散」或「汇聚」的程度）

$div F = \nabla \cdot F = \partial F_1/\partial x + \partial F_2/\partial y + \partial F_3/\partial z$

旋度（Curl）： 向量 → 向量（某点「旋转」的趋势）

$curl F = \nabla \times F = \langle\partial F_3/\partial y - \partial F_2/\partial z, \partial F_1/\partial z - \partial F_3/\partial x, \partial F_2/\partial x - \partial F_1/\partial y\rangle$

记忆技巧：三阶行列式 $det|i j k; \partial/\partial x \partial/\partial y \partial/\partial z; F_1 F_2 F_3|$ 展开即可。

Laplacian： 标量 → 标量

$Δf = div(\nabla f) = \partial^2f/\partial x^2 + \partial^2f/\partial y^2 + \partial^2f/\partial z^2$

DEFINITION — Divergence / 散度

Let $W \subseteq \mathbb{R}^3$ and let $u : W \to \mathbb{R}^3$, $u(x, y, z) = u_1(x, y, z)i + u_2(x, y, z)j + u_3(x, y, z)k$, denote a $C^1$ vector field. The divergence of $u$ is

$div u = \nabla \cdot u = \partial u_1/\partial x + \partial u_2/\partial y + \partial u_3/\partial z$.

DEFINITION — Curl / 旋度

The curl of a $C^1$ vector field $F = F_1i + F_2j + F_3k$ is

$curl F = \nabla \times F = (\partial F_3/\partial y - \partial F_2/\partial z)i + (\partial F_1/\partial z - \partial F_3/\partial x)j + (\partial F_2/\partial x - \partial F_1/\partial y)k$.

THEOREM — 两个重要恒等式

(1) $curl(\nabla f) = 0$ —— 梯度的旋度总是零（无旋条件）

(2) $div(curl F) = 0$ —— 旋度的散度总是零（无源条件）

这两条恒等式物理意义深刻，也是考试中证明题的基础。记住它们的口诀：「梯无旋，旋无散」。

第三部分：线积分

线积分的计算

给定向量场 $F$ 和参数化曲线 $r(t), a \le t \le b$：

$\int_C F \cdot dr = \int_a^b F(r(t)) \cdot r'(t) dt$

符号解释： $dr$ 是曲线的有向线元（oriented line element），等于 $r'(t) dt$。物理上，如果 $F$ 是力场，$\int_C F\cdotdr$ 就是力沿曲线做的功（work）。

闭曲线（closed curve）上的线积分称为环量（circulation），记为 $\oint_C F\cdotdr$。

例：Gradient 场的线积分（19-Q8 相关）

计算 $\int_C (f\nabla g + g\nabla f)\cdotdr$，其中 $C$ 是椭圆 $x^2/a^2 + y^2/b^2 = 1$（逆时针方向），从 $(a,0)$ 开始到 $(0,b)$。

关键观察： $f\nabla g + g\nabla f = \nabla(fg)$（乘积法则的梯度形式）。
所以 $F = \nabla(fg)$ 是保守场！线积分 = $fg|_A^B = f(B)g(B) - f(A)g(A)$。
不需要真的沿着椭圆做积分——这就是保守场的威力。

第四部分：Green 定理

GREEN'S THEOREM / 格林定理

Let $C$ be a positively oriented, piecewise smooth, simple closed curve in the plane, and let $D$ be the region bounded by $C$. If $P$ and $Q$ are functions of $(x, y)$ defined on an open region containing $D$ and have continuous partial derivatives there, then

$\oint_C P dx + Q dy = \iint_D (\partial Q/\partial x - \partial P/\partial y) dA$.

等号左边：绕边界一圈的线积分；右边：边界内部某个量的二重积分。

物理直觉

想象你绕着一个小湖走一圈，记录下风速沿你行走方向的分量。Green 定理说：这些值的总和等于整个湖面上风的「旋度」的积分。如果湖面上每个点都没有旋转（$\partial Q/\partial x - \partial P/\partial y = 0$），绕一圈总做功就是 0——这也恰好是保守场的特征。

Green 定理的常见应用

化线积分为面积分： 如果被积函数 $P, Q$ 使得 $\partial Q/\partial x - \partial P/\partial y$ 很简单，可以绕过直接做线积分。
用线积分求面积：

$Area(D) = \oint_C x dy = -\oint_C y dx = \frac{1}{2} \oint_C (x dy - y dx)$

三个公式等价，选最方便的那个用。

寻找最大环量的闭曲线： 给定 $curl$ 的分布，找到使 $\oint F\cdotdr$ 最大的闭曲线——这是 19-Q10c 的考点。

例：用 Green 定理计算线积分（19-Q10b）

计算 $\oint_C (x^2 - y) dx + (x + y^2) dy$，其中 $C$ 是单位圆 $x^2 + y^2 = 1$（逆时针）。

Step 1： 识别 $P = x^2 - y, Q = x + y^2$。
Step 2： 求偏导：$\partial Q/\partial x = 1$，$\partial P/\partial y = -1$。
Step 3： $\partial Q/\partial x - \partial P/\partial y = 1 - (-1) = 2$。
Step 4： Green 定理：$\oint = \iint 2 dA = 2 \times Area(D) = 2\pi$。
一步到位！ 如果不借助 Green 定理，需要参数化圆 $x = cos t, y = sin t$，做一次冗长的一维积分——显然 Green 定理更优雅。

第五部分：Stokes 定理

STOKES' THEOREM / 斯托克斯定理

Let $S$ be a piecewise smooth oriented surface in $\mathbb{R}^3$, whose boundary $\partial S$ is a piecewise smooth, simple, closed curve, with orientation induced from that of $S$. If $F$ is a $C^1$ vector field defined on an open region containing $S$, then

$\oint_{\partial S} F \cdot dr = \iint_S (curl F) \cdot n dS$.

边界曲线上的环量 = 曲面上旋度法向分量的面积分。

Stokes 定理的核心地位

注意：Green 定理实际上是 Stokes 定理的特例——当 $S$ 是 xy 平面上的一块平面区域时，$(curl F)\cdot n = \partial F_2/\partial x - \partial F_1/\partial y$（取 $n = k$），Stokes 定理就退化为 Green 定理。

Stokes 定理告诉我们一个深刻的物理事实：向量场绕边界一周的环量，完全由内部每一点的旋度决定。这也解释了为什么保守场（$curl F = 0$）的环量为 0——内部没有旋度，「转不起来」。

用 Stokes 定理证明 curl(∇f) = 0（19-Q11c）

Step 1： 任取一条闭曲线 $C$，它是某曲面 $S$ 的边界。
Step 2： 由 Stokes 定理：$\oint_C \nabla f \cdot dr = \iint_S curl(\nabla f)\cdot n dS$。
Step 3： 但 $\nabla f$ 是保守场：$\int_C \nabla f\cdotdr = f(终点) - f(起点) = 0$（闭曲线）。
Step 4： 所以 $\iint_S curl(\nabla f)\cdot n dS = 0$ 对任意曲面 $S$ 成立。这意味着 $curl(\nabla f) = 0$。

环量密度的物理意义（21-Q2c）

公式 $(curl F)\cdot n = \lim_{A \to 0} (1/A) \oint_C F\cdotdr$（其中边界 $C$ 围成的面积为 $A$，n 是法向）。

这意味着 $(curl F)\cdot n$ 是单位面积上的环量，即「该方向上的旋转密度」。一个扁平的叶片放在流场中，旋转最快的那个方向就是 $curl F$ 的方向。

右手法则： Stokes 定理中，曲面的法向 $n$ 和边界 $C$ 的方向必须满足右手法则——如果你的右手四指沿着 $C$ 的方向弯曲，大拇指所指的方向就是 $n$。考试中经常要求你明确说明方向的选择，不写会扣分。

第六部分：Gauss 散度定理

GAUSS' DIVERGENCE THEOREM / 高斯散度定理

Let $D$ be a connected region in $\mathbb{R}^3$ whose boundary $S = \partial D$ is a piecewise smooth closed orientable surface. Let $n$ be the outward pointing unit normal vector field of $S$. If $F$ is a $C^1$ vector field defined on an open region containing $D$, then

$\iint_S F \cdot n dS = \iiint_D div F dV$.

闭合曲面上的净通量（flux）= 内部散度的体积分。

物理解释

想象 $F$ 是水流的速度场。散度定理说：单位时间内「流出闭合曲面 S 的总水量」=「S 内部所有水源的产水率之和」。如果内部没有水源（$div F = 0$），那么流进去的必定等于流出来的——净通量为零。

散度定理的经典题型

通量计算（19-Q12b, 25-Q4）： 直接算曲面积分可能涉及多个面片，每个面片还要单独参数化。而散度定理把曲面上的积分转化为体积内的积分——往往简单很多。
恒等式证明（19-Q12c）： $div(fF) = f div(F) + \nabla f \cdot F$。这是考试中常见的证明题起点。
体积计算： 取 $F = \langle x, 0, 0\rangle$，则 $div F = 1$，$Volume(D) = \iiint 1 dV = \oiint x n_1 dS$。换言之，体积可以表示为边界上的曲面积分——跟 Green 定理求面积是同一个思路。

例：通量计算（19-Q12b 风格）

计算向量场 $F = \langlex, y, z\rangle$ 通过单位球面 $x^2+y^2+z^2 = 1$ 的向外通量。

Step 1： 球面是封闭的 → 散度定理适用。
Step 2： $div F = \partial x/\partial x + \partial y/\partial y + \partial z/\partial z = 1+1+1 = 3$。
Step 3： $\oiint F\cdot n dS = \iiint 3 dV = 3 \times Volume(单位球) = 3 \times (4\pi/3) = 4\pi$。

第七部分：三大定理的总览与考试应对

三大定理的递进关系

Green 定理（2D 平面）： 闭曲线上的环量 = 区域内 (∂Q/∂x - ∂P/∂y) 的面积分
$\oint_\partial D ω = \iint_D dω$
Stokes 定理（3D 曲面）： 闭曲线上的环量 = 曲面上 curl 的面积分
$\oint_{\partial S} F\cdotdr = \iint_S curl F\cdot n dS$
散度定理（3D 体积）： 闭曲面上的通量 = 体积内 div 的体积分
$\oiint_\partial D F\cdot n dS = \iiint_D div F dV$

这三个定理本质上都是推广的微积分基本定理（Generalized Stokes Theorem）：$\int_\partialΩ ω = \int_Ω dω$——边界上的积分 = 内部的微分。

考试中如何选择定理？
  • 闭曲线 + 平面 → Green 定理。 关键词：xy-平面上的闭曲线，线积分。
  • 闭曲线 + 空间曲面 → Stokes 定理。 关键词：空间曲线，曲面，环量。
  • 闭曲面 → 散度定理。 关键词：封闭曲面，通量，外法向。
  • 保守场 → 基本定理。 关键词：线积分与路径无关，curl F = 0。

例：判断保守场（19-Q7b-i）

判断 $F = 4xz i + 6y j + 84z k$ 是否为保守场。

Step 1： 计算 curl：
$curl_x = \partial F_3/\partial y - \partial F_2/\partial z = \partial(84z)/\partial y - \partial(6y)/\partial z = 0 - 0 = 0$
$curl_y = \partial F_1/\partial z - \partial F_3/\partial x = \partial(4xz)/\partial z - \partial(84z)/\partial x = 4x - 0 = 4x$
$curl_z = \partial F_2/\partial x - \partial F_1/\partial y = \partial(6y)/\partial x - \partial(4xz)/\partial y = 0 - 0 = 0$
Step 2： $curl F = \langle0, 4x, 0\rangle \neq 0$。所以 $F$ 不是保守场。
检查： 如果题目是 $F = 4xz i + 6y j + 2x^2 k$ 呢？那 $curl_y = \partial(4xz)/\partial z - \partial(2x^2)/\partial x = 4x - 4x = 0$，curl 全为零 → 保守场，且在整个 $\mathbb{R}^3$（单连通）上定义 → $F$ 是保守场。

课后测验

Q1：关于三个向量微积分定理，以下哪项对应关系是错误的？
A) Green → 闭曲线环量 = 平面区域上的二重积分
B) Stokes → 闭曲线环量 = 曲面上的旋度面积分
C) Gauss → 闭曲面通量 = 闭曲面内的旋度体积分
D) 三个定理都是 ∫_∂Ω ω = ∫_Ω dω 的特例
解析： 散度（Gauss）定理说的是闭曲面上的通量 = 内部 div（散度）的体积分，不是旋度的体积分。旋度出现在 Stokes 定理里。选 C。
Q2：设 $F = \langle2xy, x^2\rangle$，以下说法正确的是？
A) F 是保守场（在整个 ℝ² 上），势函数 φ = x²y
B) F 不是保守场，因为 curl F ≠ 0
C) F 是保守场，但只能在不包含原点的区域上
D) F 是保守场，势函数 φ = xy²
解析： $P=2xy, Q=x^2$。验证：$\partial P/\partial y = 2x$，$\partial Q/\partial x = 2x$，相等。且 $\mathbb{R}^2$ 是单连通区域，所以 $F$ 是保守场。找势函数：$\partialφ/\partial x = 2xy \to φ = x^2y + g(y)$；$\partialφ/\partial y = x^2 + g'(y) = x^2 \to g'(y)=0 \to g(y)=C$。所以 $φ = x^2y + C$。选 A。
Q3：用 Green 定理计算 $\oint_C (e^x - y^3) dx + (cos y + x^3) dy$，其中 $C$ 是逆时针方向的单位圆 $x^2 + y^2 = 1$。结果是什么？
A) 0
B) 2π
C) 3π/2
D) π
解析： $P = e^x - y^3, Q = cos y + x^3$。
$\partial Q/\partial x = 3x^2$，$\partial P/\partial y = -3y^2$。
$\partial Q/\partial x - \partial P/\partial y = 3x^2 - (-3y^2) = 3(x^2+y^2)$。
$\oint = \iint 3(x^2+y^2) dA$。换到极坐标：$3\iint r^2 \cdot r dr dθ = 3\int_0^{2\pi} dθ \int_0^1 r^3 dr = 3\cdot2\pi\cdot(1/4) = 3\pi/2$。选 C。
Q4：对于一个在 $\mathbb{R}^3$ 中光滑的向量场 $F$，恒等式 $div(curl F) = 0$ 可以通过哪个定理来理解？
A) Green 定理
B) Fubini 定理
C) 结合 Stokes 定理和散度定理
D) 微积分基本定理
解析： 对任意闭曲面 $S$（边界为空），由散度定理和 Stokes 定理：
$\iiint_D div(curl F) dV = \oiint_S (curl F)\cdot n dS$。
但 $S$ 是闭曲面，可以「从中切开」成两片，每片的边界方向相反 → 净环量为 0 → $\oiint_S curl F\cdot n dS = 0$。因此 $\iiint div(curl F) dV = 0$ 对任意区域成立 → $div(curl F) = 0$。选 C。

本课总结 — 三大定理速记表

  • Green 定理： $\oint_C Pdx+Qdy = \iint_D (\partial Q/\partial x-\partial P/\partial y) dA$。平面闭曲线环量 → 面积分。
  • Stokes 定理： $\oint_C F\cdotdr = \iint_S curl F\cdot n dS$。空间闭曲线环量 → 曲面积分。
  • 散度定理： $\oiint_S F\cdot n dS = \iiint_D div F dV$。闭曲面通量 → 体积分。
  • 保守场判定： 2D 看 $\partial P/\partial y = \partial Q/\partial x$；3D 看 $curl F = 0$。记得加「单连通」的前提。
  • 核心恒等式： $curl(\nabla f) = 0$（梯无旋），$div(curl F) = 0$（旋无散）
  • 考试策略： 先判断积分类型 → 选择合适的定理 → 对应公式代入 → 计算。30-40% 的分数在这里！
