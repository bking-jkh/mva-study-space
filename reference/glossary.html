MVA 术语表 Glossary

一、基础概念 Foundations

function of several variables

多元函数 — 定义域是 Rⁿ 的子集、值域是 R 的子集的函数。记为 f: D ⊆ Rⁿ → R。例如 f(x,y) = x² + y²。

level curve / contour

水平曲线 / 等高线 — 满足 f(x,y) = c（c 为常数）的所有点 (x,y) 的集合。在拓扑地图中表示相同海拔的连线。

vertical section

垂直截面 — 固定一个自变量，看函数随另一个变量变化的图形。在 x₀ 处沿 y 方向的垂直截面是 y ↦ f(x₀, y) 的图像。

open n-dimensional ball

开球 — Bⁿ(x, r) = {y ∈ Rⁿ : |x − y| < r}，以 x 为中心、r 为半径但不包含球面的 n 维球体。

interior point / boundary point

内点 / 边界点 — 如果一个点存在一个完全包含在集合内的开球邻域，则为内点；如果在闭包中但不是内点，则为边界点。

二、极限与连续 Limits & Continuity

limit of f as x approaches w

多元极限 — lim_{x→w} f(x) = L 指：对任意 ε > 0，存在 δ > 0，使得 0 < |x − w| < δ 时 |f(x) − L| < ε。注意：在 Rⁿ 中可以从无穷多个方向逼近。

continuity

连续性 — 如果 lim_{x→w} f(x) = f(w)，则称 f 在 w 处连续。

三、偏导数 Partial Derivatives

partial derivative ∂f/∂xⱼ

偏导数 — 将其他变量视为常数，只对一个变量求导。定义：∂f/∂xⱼ(w) = lim_{h→0} [f(w₁,...,wⱼ+h,...,wₙ) − f(w)] / h。

mixed second order partial derivatives

二阶混合偏导数 — ∂²f/∂xᵢ∂xⱼ，先对 xⱼ 求偏导再对 xᵢ 求偏导。

Clairaut's Theorem

克莱罗定理 — 如果混合偏导数在某点连续，则求导次序可交换：∂²f/∂x∂y = ∂²f/∂y∂x。

四、链式法则 Chain Rule

chain rule

链式法则 — ∂f/∂tᵢ = (∂f/∂x₁)(∂x₁/∂tᵢ) + ... + (∂f/∂xₙ)(∂xₙ/∂tᵢ)。用树状图记忆。

implicit differentiation

隐函数求导 — 若 f(x,y) = 0 隐式定义 y 为 x 的函数，则 dy/dx = −(∂f/∂x)/(∂f/∂y)。

五、方向导数与梯度 Directional Derivatives & Gradient

directional derivative Dᵤf

方向导数 — 函数在给定方向 u 上的变化率。Dᵤf(x) = lim_{h→0} [f(x + hû) − f(x)] / h。

gradient vector ∇f

梯度向量 — ∇f(a) = ⟨∂f/∂x₁(a), ..., ∂f/∂xₙ(a)⟩。梯度的方向是函数增长最快的方向，|∇f| 是最大变化率。

Dᵤf = ∇f · u

方向导数等于梯度与单位方向向量的点积：Dᵤf(a) = ∇f(a) · û。

tangent plane

切平面 — 曲面在一点处的切平面以梯度为法向量。方程：⟨x − a, y − b, z − c⟩ · ∇g(a,b,c) = 0。

六、极值与 Hessian Extrema & Hessian

stationary point / critical point / singular point

驻点（∇f = 0）/ 临界点（驻点或奇点）/ 奇点（∇f 不存在）。

saddle point

鞍点 — 驻点但不是局部极值点。在某些方向上是极大，在另一些方向上是极小。

Hessian matrix

Hessian 矩阵 — 所有二阶偏导数组成的方阵。用于判定驻点性质（主 minor 判别法）。

leading minor test

主 minor 判别法 — 对 n=2：若 det(H₁) < 0, det(H) > 0 → 极大；det(H₁) > 0, det(H) > 0 → 极小；否则为鞍点。

七、Lagrange 乘子 Lagrange Multipliers

Lagrange multiplier λ

拉格朗日乘子 — 在有约束 g(x) = 0 的条件下求 f 的极值时，构造 Lagrangian L = f + λg，然后令 ∇L = 0。

Lagrangian

拉格朗日函数 — L(x₁,...,xₙ, λ₁,...,λₖ) = f(x) + Σ λᵢ gᵢ(x)。

八、多重积分 Multiple Integrals

double integral ∬ f dA

二重积分 — 二元函数在平面区域上的积分，表示曲面下的体积（当 f ≥ 0）。

triple integral ∭ f dV

三重积分 — 三元函数在空间区域上的积分，可表示密度函数的质量。

Fubini-Tonelli Theorem

富比尼-托内利定理 — 允许将重积分写成累次积分（repeated integral），且可以交换积分次序。

Riemann sum

黎曼和 — 将区域划分为小矩形/长方体，取样本点，求和。当划分趋于零时即得积分值。

九、坐标变换 Coordinate Transformations

polar coordinates (r, θ)

极坐标 — x = r cos θ, y = r sin θ, dA = r dr dθ。

cylindrical polar coordinates (r, θ, z)

柱坐标 — x = r cos θ, y = r sin θ, z = z, dV = r dr dθ dz。

spherical polar coordinates (ρ, φ, θ)

球坐标 — x = ρ sin φ cos θ, y = ρ sin φ sin θ, z = ρ cos φ, dV = ρ² sin φ dρ dφ dθ。

Jacobian ∂(x,y)/∂(u,v)

雅可比行列式 — 坐标变换时的面积/体积缩放因子。dx dy = |∂(x,y)/∂(u,v)| du dv。

十、向量微积分 Vector Calculus

conservative vector field

保守向量场 — 存在势函数 φ 使得 F = ∇φ。等价条件：线积分与路径无关，或 curl F = 0（在单连通区域上）。

divergence div F

散度 — div F = ∂F₁/∂x + ∂F₂/∂y + ∂F₃/∂z = ∇ · F，衡量向量场在某点的"发散"程度。

curl curl F

旋度 — curl F = ∇ × F，衡量向量场在某点的"旋转"程度。

line integral ∫_C F · dr

线积分 — 向量场沿曲线的积分，可理解为力沿路径做的功。

Green's Theorem

格林定理 — ∮_C P dx + Q dy = ∬_D (∂Q/∂x − ∂P/∂y) dA，将平面闭曲线上的线积分转化为区域上的二重积分。

Stokes' Theorem

斯托克斯定理 — ∮_C F · dr = ∬_S (curl F) · n dS，将空间闭曲线上的线积分转化为曲面上的面积分。

Gauss' Divergence Theorem

高斯散度定理 — ∬_S F · n dS = ∭_D div F dV，将闭曲面上的通量积分转化为区域上的三重积分。

Laplacian Δf

拉普拉斯算子 — Δf = ∇²f = ∂²f/∂x² + ∂²f/∂y² + ∂²f/∂z² = div(∇f)。满足 Δf = 0 的函数称为调和函数。

Taylor polynomial (multivariate)

多元泰勒多项式 — 多元函数在某点附近的近似展开。如 f(x,y) 在 (0,0) 的三阶 Taylor 展开。

Cobb-Douglas function

柯布-道格拉斯函数 — 生产函数 f(x,y) = Ax^α y^β，常用于经济学优化问题（Lagrange 乘子法的典型应用）。
